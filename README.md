# FeiFood – Food Ordering CLI (Python)

FeiFood is a console-based application developed in Python that simulates a food delivery app similar to iFood.
This project was created as part of academic activities during the Computer Science degree at FEI.

## Features
- User registration
- User login
- Menu visualization
- Order creation
- Order payment simulation

## Technologies
- Python

# FeiFood – CLI de pedidos de alimentos (Python)

FeiFood é um aplicativo baseado em console desenvolvido em Python que simula um aplicativo de entrega de comida semelhante ao iFood.
Este projeto foi criado no âmbito de atividades académicas durante a licenciatura em Ciência da Computação da FEI.

## Características
- Registro de usuário
- Login do usuário 
- Visualização do menu 
- Criação de pedidos
- Simulação de pagamento de pedidos

## Tecnologias 
- Python
